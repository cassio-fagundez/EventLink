## 📱 Sobre el Proyecto

**EventLink** es una aplicación móvil para Android diseñada para optimizar la difusión y descubrimiento de eventos. Este proyecto fue desarrollado como **Trabajo de Conclusión de Curso (TCC)** para la carrera binacional de **Tecnólogo en Análisis y Desarrollo de Sistemas** de **UTEC** (Uruguay) e **IFSUL** (Brasil).

Surge como respuesta a las incertezas y limitaciones de las tradicionales difusiones verbales, centralizando la información en una plataforma accesible y fácil de usar.

La aplicación permite a los usuarios:
- ✅ Crear eventos de manera sencilla
- 🔍 Explorar eventos cercanos y de interés
- ⭐ Marcar interés en eventos específicos
- 📍 Descubrir eventos usando tecnología de mapas

Desarrollada con metodologías ágiles y tecnologías modernas como Firebase y OpenStreetMap, EventLink se posiciona como una herramienta innovadora, especialmente orientada a enriquecer la interacción social y cultural en regiones como Brasil y Uruguay.

## 🚀 Características Principales

- **Creación de Eventos**: Publica tus eventos de forma rápida y sencilla
- **Descubrimiento**: Encuentra eventos relevantes cerca de tu ubicación
- **Interacción Social**: Expresa interés en eventos y conecta con otros asistentes
- **Mapas Integrados**: Visualización geográfica usando OpenStreetMap
- **Backend en la Nube**: Almacenamiento y sincronización con Firebase

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Java
- **Plataforma**: Android (API 27+)
- **IDE**: Android Studio
- **Backend**: Firebase
- **Mapas**: OpenStreetMap
- **Arquitectura**: MVVM (Model-View-ViewModel)
- **Metodología**: Desarrollo Ágil

## 📋 Requisitos del Sistema

- **Android**: API 27 (Android 8.0) o superior
- **Memoria RAM**: Mínimo 2GB recomendado
- **Espacio de Almacenamiento**: 50MB disponibles
- **Conexión a Internet**: Requerida para funcionalidades completas

## 📦 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/cassio-fagundez/EventLink.git
```

2. Abre el proyecto en Android Studio

3. Configura tu archivo `local.properties` con tus credenciales de Firebase

4. Sincroniza el proyecto con Gradle

5. Ejecuta la aplicación en un emulador o dispositivo físico

## 🔧 Configuración

1. **Firebase Setup**:
   - Crea un nuevo proyecto en Firebase Console
   - Configura Authentication y Firestore Database
   - Descarga el archivo `google-services.json` y colócalo en `app/src/main/`

2. **API Keys**:
   - Configura las claves de API necesarias en `local.properties`
   - Asegúrate de incluir las credenciales para OpenStreetMap
