## 📱 About the Project

**EventLink** is an Android mobile application designed to optimize the promotion and discovery of events. This project was developed as a **Final Course Project (TCC)** for the binational degree in **Technology in Systems Analysis and Development** from **UTEC** (Uruguay) and **IFSUL** (Brazil).

It emerges as a response to the uncertainties and limitations of traditional verbal event dissemination, centralizing information on an accessible and easy-to-use platform.

The application allows users to:
- ✅ Create events easily
- 🔍 Explore nearby and interesting events
- ⭐ Mark interest in specific events
- 📍 Discover events using map technology

Developed with agile methodologies and modern technologies such as Firebase and OpenStreetMap, EventLink positions itself as an innovative tool, especially aimed at enriching social and cultural interaction in regions like Brazil and Uruguay.

## 🚀 Key Features

- **Event Creation**: Publish your events quickly and easily
- **Discovery**: Find relevant events near your location
- **Social Interaction**: Express interest in events and connect with other attendees
- **Integrated Maps**: Geographic visualization using OpenStreetMap
- **Cloud Backend**: Storage and synchronization with Firebase

## 🛠️ Technologies Used

- **Language**: Java
- **Platform**: Android (API 27+)
- **IDE**: Android Studio
- **Backend**: Firebase
- **Maps**: OpenStreetMap
- **Architecture**: MVVM (Model-View-ViewModel)
- **Methodology**: Agile Development

## 📋 System Requirements

- **Android**: API 27 (Android 8.0) or higher
- **RAM Memory**: Minimum 2GB recommended
- **Storage Space**: 50MB available
- **Internet Connection**: Required for full functionality

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/cassio-fagundez/EventLink.git
```

2. Open the project in Android Studio

3. Configure your `local.properties` file with your Firebase credentials

4. Sync the project with Gradle

5. Run the application on an emulator or physical device

## 🔧 Configuration

1. **Firebase Setup**:
   - Create a new project in Firebase Console
   - Set up Authentication and Firestore Database
   - Download the `google-services.json` file and place it in `app/src/main/`

2. **API Keys**:
   - Configure the necessary API keys in `local.properties`
   - Make sure to include OpenStreetMap credentials
