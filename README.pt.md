## 📱 Sobre o Projeto

**EventLink** é um aplicativo móvel Android desenvolvido para otimizar a promoção e descoberta de eventos. Este projeto foi desenvolvido como **Trabalho de Conclusão de Curso (TCC)** para o curso binacional de **Tecnologia em Análise e Desenvolvimento de Sistemas** da **UTEC** (Uruguai) e **IFSUL** (Brasil).

Surge como resposta às incertezas e limitações das tradicionais divulgações verbais de eventos, centralizando informações em uma plataforma acessível e fácil de usar.

O aplicativo permite aos usuários:
- ✅ Criar eventos de forma simples
- 🔍 Explorar eventos próximos e de interesse
- ⭐ Marcar interesse em eventos específicos
- 📍 Descobrir eventos usando tecnologia de mapas

Desenvolvido com metodologias ágeis e tecnologias modernas como Firebase e OpenStreetMap, EventLink se posiciona como uma ferramenta inovadora, especialmente voltada para enriquecer a interação social e cultural em regiões como Brasil e Uruguai.

## 🚀 Características Principais

- **Criação de Eventos**: Publique seus eventos de forma rápida e simples
- **Descoberta**: Encontre eventos relevantes próximos à sua localização
- **Interação Social**: Expresse interesse em eventos e conecte-se com outros participantes
- **Mapas Integrados**: Visualização geográfica usando OpenStreetMap
- **Backend na Nuvem**: Armazenamento e sincronização com Firebase

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: Java
- **Plataforma**: Android (API 27+)
- **IDE**: Android Studio
- **Backend**: Firebase
- **Mapas**: OpenStreetMap
- **Arquitetura**: MVVM (Model-View-ViewModel)
- **Metodologia**: Desenvolvimento Ágil

## 📋 Requisitos do Sistema

- **Android**: API 27 (Android 8.0) ou superior
- **Memória RAM**: Mínimo 2GB recomendado
- **Espaço de Armazenamento**: 50MB disponíveis
- **Conexão com Internet**: Requerida para funcionalidades completas

## 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/cassio-fagundez/EventLink.git
```

2. Abra o projeto no Android Studio

3. Configure seu arquivo `local.properties` com suas credenciais do Firebase

4. Sincronize o projeto com Gradle

5. Execute o aplicativo em um emulador ou dispositivo físico

## 🔧 Configuração

1. **Firebase Setup**:
   - Crie um novo projeto no Firebase Console
   - Configure Authentication e Firestore Database
   - Baixe o arquivo `google-services.json` e coloque-o em `app/src/main/`

2. **API Keys**:
   - Configure as chaves de API necessárias em `local.properties`
   - Certifique-se de incluir as credenciais para OpenStreetMap
